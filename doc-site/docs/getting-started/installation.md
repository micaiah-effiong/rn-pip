# Installation

You can install Rn-pip via [NPM](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)

```bash
# using npm
$ npm install rn-pip

# using pnpm
$ pnpm add rn-pip

# using yarn
$ yarn add rn-pip

```

:::tip
 Do you want to try it out before you download? Try out a simple [Demo](/docs/getting-started/playground)
 ::::
